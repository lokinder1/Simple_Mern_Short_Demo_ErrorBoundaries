# Simple_MERN_Short_Demo_ErrorBoundaries
It's a simple example or demo project to show Error Boundaries usage
it's made by using following technologies
- ReactJS
- NodeJS
- MongoDB
- ExpressJS
- MaterialUI

## How To Run 


- Client    

    1. Move To Client Directory
    cd Client/

    2. Install Packages
    yarn

    3. start Server App
    yarn start
